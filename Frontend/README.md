# Frontend

This folder contains all source code and assets for the browser extension's user interface.

## Structure
- `manifest.json` — Extension manifest (Chrome Manifest V3)
- `popup.html`/`popup.js` — UI and script for browser popup
- `content.js` — (if needed) Content scripts to interact with web pages
- `styles/` — CSS files
